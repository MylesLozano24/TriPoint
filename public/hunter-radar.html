Flagship product

Location awareness where cellular service ends.

Hunter Radar is an off-grid outdoor safety and communication system being developed for hunters, landowners, hikers, guides, and outdoor groups.

Join the Beta How it works
The problem

Groups lose track of each other fast.

Cellular service is unreliable in rural and wilderness areas, so phone-based location sharing stops working the moment it's needed most. Traditional radios provide voice but no location. Satellite communicators work well but are often expensive and subscription-based. Meanwhile, group members spread out over large properties with no way to see where anyone else actually is.

The solution

A dedicated radio network between your group.

Hunter Radar combines portable hardware with a digital map interface to help group members maintain awareness of one another while operating across large or remote properties — without requiring users to depend entirely on cell towers.

  • Group location sharing
  • Offline communication
  • Last-known location tracking
  • Short status updates
  • Emergency alerts
  • Property boundaries
  • Shared waypoints
  • Offline map access
How it works

Connect. Communicate. View.

01 — Connect

Connect

Each user carries a Hunter Radar device connected to the local interface.

02 — Communicate

Communicate

Hunter Radar devices exchange compact information through long-range radio communication.

03 — View

View

Received locations, statuses, and alerts are displayed through the Hunter Radar map interface.

User Device → LoRa Network → Hunter Radar Device → Map Interface

Current prototype

Where development stands today.

TriPoint has built and tested a working Hunter Radar prototype, combining long-range radio hardware with a developing map interface. The team is actively expanding the system's capabilities and refining the hardware ahead of broader field testing.

What's ahead

Beta testing

Try the beta test interface.

The beta website connects to prototype hardware over Bluetooth or USB, displays live and simulated location data on a map, and records diagnostic information for the development team. It's a testing tool first — built for visibility into raw data, not simplicity.
